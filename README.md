## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Visit `http://localhost:5173` to view the portfolio.

### Build

```bash
npm run build
```

The production build will be in the `dist` directory.

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- JavaScript (ES6+)

## Customization

Edit the component files in `src/components/` to customize:
- Personal information
- Project details
- Skills and technologies
- Contact information
- Social media links
